1. WHAT IS MATPLOTLIB ?

-->     Matplotlib is a Python library used for creating graphs and charts. Think of it like a tool that helps you draw pictures to show data in a visual way, like bar charts, line graphs, pie charts, and more. It's widely used by people to make their data easier to understand by turning numbers into colorful, easy-to-read images.
-->     In short, Matplotlib is like a drawing tool for data.
